# User Stories

1. As a customer, I want to be notified if my order is likely to be delayed, so that I can plan accordingly.
2. As a warehouse team member, I want to be informed of which orders are at risk of being delayed, so that I can process them in the appropriate priority order.
3. As a warehouse supervisor, I want to view all at-risk orders before dispatch, so that I can make informed prioritisation decisions without impacting other orders.
4. As an operations manager, I want a dashboard showing delivery performance by warehouse location and delivery method, so that I can see where performance drops.
5. As a customer support officer, I want visibility of all orders flagged as at risk, so that I can proactively contact affected customers before they lodge a complaint.
6. As an operations manager, I want to know which factors are associated with delay, so that I can direct my team toward the causes that matter.
